# 📊 At-Bat Analytica (野球打撃分析アプリ - React版)

「感覚をデータに変え、次の練習を明確にする」をコンセプトにした、React + TypeScript製の野球打撃セルフ分析Webアプリケーションです。
スマートフォンでの快適な操作性を追求し、電波の届きにくいグラウンドでもスタンドアロン（ローカル保存）でサクサク動作する設計を採用しています。

## 🎯 プロジェクトの目的
1. **プロセス分析**: 打率などの最終結果だけでなく、選球眼（ボール球スイング率）や打撃内容（打球の質・方向）を可視化する。
2. **簡単入力**: 1タップで入力が完結する直感的なグリッドUIを構築し、入力の手間を極限まで減らす。

## 🛠️ 技術スタック
* **Framework**: React (Vite) + TypeScript
* **Styling**: Tailwind CSS
* **Icons**: Lucide React
* **Effects**: canvas-confetti (目標達成時のお祝い演出用)
* **Storage**: LocalStorage (オフライン・サーバーレス対応)
